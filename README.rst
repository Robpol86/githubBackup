============
githubBackup
============

Backup all of your GitHub repos (with issues/wikis) and Gists.

* Golang 1.7 supported on Linux, OS X, and Windows.

Quickstart
==========

Install:

.. code:: bash

    go get https://github.com/Robpol86/githubBackup

Usage:

.. code:: bash

    githubBackup --help
