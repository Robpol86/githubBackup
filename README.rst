============
githubBackup
============

Backups all of your GitHub public/private repos (with issues/wikis) and Gists.

* Golang 1.7 supported on Linux and OS X.

Quickstart
==========

Install:

.. code:: bash

    brew install libgit2
    go get https://github.com/Robpol86/githubBackup

Usage:

.. code::

    githubBackup --help
